3DGAN CERN use caseÔÉÅ

The first interTwin use case integrated with itwinai framework is the DT for fast particle detector simulation. 3D Generative Adversarial Network (3DGAN) for generation of images of calorimeter depositions. This project is based on the prototype 3DGAN model developed at CERN and is implemented on PyTorch Lightning framework.
